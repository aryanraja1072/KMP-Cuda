# Genome Data
To download dataset: 
  * `chmod +x downloadFasta.sh`
  * `./downloadFasta.sh`

To generate data:
  * `./genData <length> <output_filename>`
  
  Defaults:
  * length = 100
  * output_filename = "data.txt"

# HDFS Log Data

Run these commands:  
* `chmod +x downloadLogs.sh`
* `./downloadLogs.sh`
